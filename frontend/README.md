Frontend setup 

### setup instruction

1. clone project
```
    git clone https://github.com/Geetika991/frontend-project.git
````
2. move into directory 

```
cd frontend
```

3. install dependencies
```
npm i 
```

4. run the server
```
npm run dev

```
